Note: Go Lang

27 May 2015

  • Basic

    • 在 Go 中，首字母大写的名称是被导出的
    • 命名返回值
    • 函数外的每个语句都必须以关键字开始（ var 、 func 、等等）， := 结构不能使用在函数外。
    • 变量在定义时没有明确的初始化时会赋值为 零值 。
    • 常量不能使用 := 语法定义。 const Pi = 3.14

  • Control Flow

    • 跟 for 一样， if 语句可以在条件之前执行一个简单的语句。由这个语句定义的变量的作用域仅在 if 范围之内。

      if v := math.Pow(x, n); v < lim {
    return v
}

    • defer 语句会延迟函数的执行直到函数返回, 按照后进先出的顺序调用被延迟的函数调用(栈)。

      func main() {
    defer fmt.Println("world")
    fmt.Println("hello")
}
  • Complex
    • Go有指针 nil 与 C 不同，Go 没有指针运算。
  • Interface
    • 不能对来自其他包的类型或基础类型定义方法
    • 返回值拷贝？
